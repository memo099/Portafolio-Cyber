# Google Cybersecurity Professional Certificate

📅 **Fecha de finalización:** Febrero 2026  
🏫 **Plataforma:** Coursera / Google  
🎓 **Tipo:** Certificación profesional (9 cursos)

## Descripción
Certificación profesional desarrollada por Google, enfocada en preparar perfiles de entrada para roles en ciberseguridad.  
Incluye práctica técnica, laboratorios y evaluaciones aplicadas.

## Competencias adquiridas

- Fundamentos de ciberseguridad y gestión de riesgos
- Identificación de amenazas, vulnerabilidades y activos
- Análisis de logs y eventos de seguridad
- Fundamentos de SIEM e IDS
- Linux aplicado a seguridad
- SQL aplicado a análisis de datos de seguridad
- Automatización básica de tareas de seguridad con Python
- Detección y respuesta a incidentes
- Preparación para roles SOC / Blue Team junior

## Cursos completados

- Foundations of Cybersecurity  
- Play It Safe: Manage Security Risks  
- Connect and Protect: Networks and Network Security  
- Tools of the Trade: Linux and SQL  
- Assets, Threats, and Vulnerabilities  
- Sound the Alarm: Detection and Response  
- Automate Cybersecurity Tasks with Python  
- Put It to Work: Prepare for Cybersecurity Jobs  
- Accelerate Your Job Search with AI
